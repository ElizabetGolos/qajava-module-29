1. Метод isTeenager имеет ошибки. Чтобы метод isTeenager() возвращал true, если в метод передаётся значение,
 соответствующее возрасту тинейджеров — тех, кому от 13 до 19 лет (thirteen и nineteen), необходимо исправить
 условие таким образом
 вместо:
 if (age < 19 )


   необходимо писать:
 if (age <= 19 && age >=13)


Обнаруженные ошибки при значениях: -1, 0, 12.
Ожидаемый результат: false
Актуальный результат: true

Обнаруженные ошибки при значени 19:
Ожидаемый результат: true
Актуальный результат: false

 После иправления метода isTeenager() тесты выполняются без ошибок
